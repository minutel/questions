# 智能课程规划助手

大一C++课程大作业，模拟AI选课顾问。

## 功能
- 解析课程先修关系
- 基于兴趣标签推荐课程
- 在学分约束下生成最优课表
## courses.txt
课程列表、学分、课程前导关系、标签存储在`courses.txt`中，格式如下：
```
课程ID 课程名称 学分 先修课程ID1,先修课程ID2,... 标签1,标签2,...
```
例如：
```
CS101 程序设计基础 4  AI,编程
CS102 数据结构与算法 3 CS101 AI,算法
CS103 计算机组成 3 CS101 系统,硬件
CS201 操作系统 3 CS102,CS103 系统
CS202 数据库理论与应用 3 CS101 AI,应用
CS301 机器学习导论 3 CS102,CS202,MATH201 AI,数据科学
MATH101 数学分析 4  数学,基础
MATH102 高等代数 4  数学,AI
MATH201 概率统计 3 MATH101,MATH102 数学,AI
```
## user_progress.txt
学生学习进度、兴趣信息存储在`user_progress.txt`中，格式如下：
```
学生ID
已修课程：S001,S002,...
兴趣标签：AI, 数据科学
当前学期：3
```
例如：
```
U1001
已修课程: CS101,MATH201
兴趣标签: AI, 数据科学
当前学期: 3
```
